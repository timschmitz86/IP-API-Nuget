# IP-API-Nuget

[![.NET](https://github.com/timschmitz86/IP-API-Nuget/actions/workflows/dotnet.yml/badge.svg?branch=main)](https://github.com/timschmitz86/IP-API-Nuget/actions/workflows/dotnet.yml)

Small nuget package written in C#.
It just enables you to use [IPAPI.co](https://IPAPI.co) in native C#.

Please keep in mind [IPAPI.co](https://IPAPI.co) is only for free if you use it for private reasons.
For enterprise usage please check license of provider IPAPI.co.
